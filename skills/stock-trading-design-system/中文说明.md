# stock-trading-design-system 中文说明

## 这个 Skill 是什么

`stock-trading-design-system` 是股票交易 App 的设计系统规范 Skill。

它的核心作用是约束页面和组件输出，让新页面、新模块或原型尽量符合当前 App 的视觉语言、信息密度和组件规则。

## 适合什么时候用

- 审核页面是否符合当前设计系统。
- 约束页面结构、组件命名和视觉规则。
- 检查颜色、字号、圆角、间距、列表密度是否跑偏。
- 生成或修改股票交易类页面前，先确定设计系统边界。
- 配合原型生成 Skill 使用，提高还原度。

## 不适合做什么

- 不单独写完整 PRD。
- 不直接负责生成完整 HTML 原型。
- 不用于营销页、通用网站或非股票交易 App 页面。

## 在当前项目中的定位

它是设计系统约束层。它告诉后续原型生成：页面应该长什么样、组件应该怎么组织、哪些视觉规则不能随意改变。
