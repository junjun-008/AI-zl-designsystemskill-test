# 仪表盘列表页骨架

## 适用场景

- `Aktivitas`
- 紧凑型投资组合概览
- 包含行情列表模块的账户首页

## 结构

1. 顶部 App Bar
2. 主要摘要区
3. 快捷操作
4. 一个或多个列表型行情模块
5. 底部导航

## 布局规则

- 屏幕宽度：`375px`
- 左右边距：`20px`
- 主要模块间距：`16px`
- 分组区块使用 Section 卡片
- 快捷操作的视觉优先级低于余额区

## 内容规则

- 首屏优先突出摘要数字。
- 列表行保持紧凑、便于扫描。
- 模块内部的次级导航使用文字按钮。

## 常用组件

- `Top App Bar`
- `Balance Summary`
- `Quick Action Button`
- `Watchlist`
- `Stock List Item`
- `Bottom Navigation`
