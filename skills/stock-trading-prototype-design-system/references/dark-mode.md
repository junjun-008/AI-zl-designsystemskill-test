# 暗色模式规则

MVP 默认生成亮色。只有当用户明确要求暗色，或要求明暗两套时，才生成暗色。

## 来源

暗色 token 来自已确认的 `视觉风格2.0 / Color` 变量和 Page 1 暗色源稿。

## 使用规则

- 不单独发明暗色值。
- 使用 `assets/design-tokens.css` 中 `[data-theme="dark"]` 的变量覆盖。
- 金融 rise/fall 在明暗色中保持语义一致。
- 暗色页面优先保留深色背景、浅色文字、蓝色激活态的源稿关系。

## 输出要求

生成暗色 HTML 原型时：

- 在根节点使用 `data-theme="dark"`。
- 保留同一页面结构，不因暗色重排页面。
- 说明暗色依据来自 token 映射。
