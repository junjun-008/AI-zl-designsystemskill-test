# 移动端布局规则

## 默认画布

- 默认宽度：390px。
- 默认主题：亮色。
- 页面以纵向滚动为主。
- 底部导航存在时，内容需要留出底部安全空间。

## 布局密度

- 当前 App 信息密度较高，避免过度留白。
- 卡片之间使用紧凑间距。
- 同一页面内不要混用太多卡片形态。

## 响应式策略

MVP 阶段优先保证 390px 原型效果。若需要适配其他宽度：

- 保持页面最大宽度 390px 或居中展示。
- 不转换为桌面双栏仪表盘。
- 不因屏幕变宽而放大字号。

## 固定元素

- 底部导航可固定在底部。
- 弹窗、抽屉、遮罩只在交互说明需要时出现。
- 不使用复杂动画作为核心表达。
