# stock-trading-prototype-design-system 中文说明

## 这个 Skill 是什么

`stock-trading-prototype-design-system` 是股票交易 App 的 HTML 原型生成 Skill。

它的核心作用是基于 PRD、Figma 页面截图和设计系统规则，生成 390px 移动端 HTML 可交互原型，用于产品评审和方案验证。

## 适合什么时候用

- 根据 PRD 生成移动端 HTML 原型。
- 在已有 Figma 页面基础上新增模块或调整功能。
- 生成可点击、可切换状态的评审用原型。
- 快速验证页面结构、信息层级和交互路径。
- 配合 Figma 截图提高现有页面还原度。

## 不适合做什么

- 不负责写 PRD。
- 不保证一次生成设计师最终精修稿。
- 不用于通用金融卡片风页面。
- 不用于后台、数据分析、营销页或桌面网页。

## 在当前项目中的定位

它是原型输出层。前面先用 `ai-zl-prdwriting` 明确需求，再用 `stock-trading-design-system` 约束视觉和组件，最后用它生成 HTML 原型。
