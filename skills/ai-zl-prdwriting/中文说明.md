# ai-zl-prdwriting 中文说明

## 这个 Skill 是什么

`ai-zl-prdwriting` 是 AI-zl 项目的 PRD 写作和需求结构化 Skill。

它的核心作用不是生成页面，而是把一个想法、功能点或页面调整需求，整理成产品、设计、研发和测试都能理解的 PRD。

## 适合什么时候用

- 写新功能 PRD。
- 重写或补充已有 PRD。
- 把口头需求整理成文档。
- 补充验收标准。
- 明确页面目标、用户场景、功能范围和边界。
- 在生成原型前先把需求讲清楚。

## 不适合做什么

- 不直接生成 HTML 原型。
- 不负责还原 Figma 视觉稿。
- 不负责创建设计系统组件。
- 不替代原型生成 Skill。

## 在当前项目中的定位

它是原型生成前的需求准备层。先用它把需求写清楚，再交给设计系统和原型生成 Skill，可以减少页面跑偏、入口不清、交互缺失等问题。
