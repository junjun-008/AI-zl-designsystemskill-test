# Skills 中文说明

这个目录集中保存当前项目使用的 Codex Skills。每个 Skill 都是一个独立文件夹，包含 `SKILL.md` 和相关参考资料。

## Skill 列表

### `ai-zl-prdwriting`

用于 PRD 写作、需求梳理、验收标准补全和 PM 到设计/研发的需求交付。适合在开始生成原型前，先把功能范围、页面目标、交互流程和边界条件写清楚。

### `stock-trading-design-system`

用于约束股票交易 App 的设计系统规则，包括页面结构、组件使用、布局密度、颜色语义、命名规范和设计系统一致性。适合在生成或审核页面时防止视觉风格跑偏。

### `stock-trading-prototype-design-system`

用于生成股票交易 App 的移动端 HTML 可交互原型。它应该结合 PRD、Figma 页面截图和设计系统规则使用，适合产品评审阶段快速产出接近当前 App 风格的页面原型。
